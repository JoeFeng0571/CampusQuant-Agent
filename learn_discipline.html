交易纪律与复盘

选股靠认知，生存靠纪律。一套完整的纪律系统包含四件事： 仓位管理、止损止盈、回撤控制、定期复盘。 一个也不能少。

1. 仓位管理三种流派

"买多少"比"买什么"更决定命运

流派 A：等权分仓（最稳健）

账户总资金 ÷ N = 单只仓位上限。新手首选。

单只仓位 ≤ 1 / N，N 为最大持仓数（通常 5-10）

流派 B：波动率加权（Risk Parity 思路）

高波动的少买，低波动的多买——让每只股贡献差不多的组合波动。

单只仓位 ∝ 1 / σ_i，再归一化

流派 C：凯利公式（半凯利版）

根据胜率和盈亏比计算"理论最优仓位"，实战用 0.5 倍。

Kelly = 胜率 − 败率/盈亏比 半凯利 = Kelly × 0.5
推荐新手从等权分仓起步。凯利公式需要你对自己的胜率有可靠估计，而这通常要 50 次以上交易样本。

2. 止损规则

没有止损的交易等于把亏损上限交给市场

类型 1：固定比例止损

入场价下跌 X% 立刻卖。X 通常 5-10%（A 股常用 8%）。

类型 2：移动止损（Trailing Stop）

跟随最新高点，止损价 = 高点 × (1 − X%)。

适合趋势交易——只要趋势没破坏，止损价一直上移。趋势反转时自动退出。

类型 3：ATR 止损

止损距离 = 入场价 − N × ATR（一般 N=2 或 3）。自适应波动率，高波动股票给更大空间。

选型建议

场景推荐止损
长期持股无 / 宽比例（-30%）/ 基本面止损
趋势跟随移动止损 或 跌破关键均线
短线 / 波段固定比例 5-8%
高波动题材ATR 止损 2-3 倍
止损规则必须在入场前写下。不要盘中临时"感觉还行，再等等"。

3. 止盈规则

盈亏比 ≥ 1.5 才能长期稳定

原则：盈亏比先行

入场前先算：如果按止损位卖，我亏多少？按目标价卖，我赚多少？

盈亏比 = 止盈距离 / 止损距离

盈亏比 ≥ 1.5 才值得交易。盈亏比 < 1 时，你需要 > 50% 胜率才能不亏——而散户平均胜率低于 40%。

止盈类型

  • 目标价止盈：到合理估值卖
  • 分批止盈：涨 50% 卖 1/3，涨 100% 再卖 1/3，剩下裸奔
  • 跟踪止盈：同移动止损，让利润奔跑
  • 基本面止盈：逻辑兑现（如业绩达标）就走

4. 最大回撤控制

回撤的不对称性比你想的可怕

三级熔断建议

  • 单笔熔断：任何一笔亏损不超过账户 2-3%
  • 月度熔断：月累计亏损达到 10% 暂停新开仓，只管理已有仓位
  • 年度熔断：年累计亏损 20% 强制清仓休息 1 月，复盘再决定
控制回撤不是为了跑不赢别人，是为了不被市场"踢出局"。活下去是复利的前提。

5. 交易日志 + 周度复盘模板

每笔交易 3 句话

买入逻辑: （1-2 句，为什么买） 止损位: （具体价格 / 相对比例） 卖出条件: （达到何时卖 / 基本面变化何时卖）

每周复盘 5 问

  1. 本周交易胜率多少？盈亏比多少？
  2. 有没有违反自己写下的规则？（仓位、止损、频率）
  3. 有没有 FOMO 驱动的交易？当时是什么触发？
  4. 本周最大盈利和最大亏损的原因是什么？
  5. 下周计划做什么，不做什么？

每月统计 5 指标

  • 胜率 = 盈利笔数 / 总笔数
  • 盈亏比 = 平均盈利 / 平均亏损
  • 期望值 = 胜率 × 平均盈利 − 败率 × 平均亏损
  • 最大回撤
  • 累计收益 vs 基准（沪深 300 / 标普 500）
3 个月的交易日志比 300 小时的选股课更能让你成长。因为它照出的是你真实的习惯。
