宏观框架：让市场快讯对你不再是天书

"美联储今晚议息"、"LPR 调降"、"社融同比回落"、"PMI 跌破枯荣线"——这些词如果你一看就懂， 市场快讯对你就是免费的投资情报。不懂的话，就只是噪音。

1. 利率如何影响股市

货币是万物之价

核心链条

利率 → 贴现率 → 资产定价。利率下降 → 未来现金流折现值提高 → 资产估值走高。 同样的公司，在 2% 利率环境下值 30PE，在 5% 利率环境下只值 15PE。

两种周期的股市特征

周期利率方向受益板块受压板块
降息周期↓成长股 / 科技 / 地产 / 高股息银行 (息差收窄)
加息周期↑银行 / 保险 / 价值股成长股 / 地产 / 新兴市场
记住"降息利好成长，加息利好价值"这个大方向，市场分析 60% 的框架已经搭好。

2. 汇率与外资流动

看懂北向资金为什么流出

基本原理

外资买 A 股需要先把外币换成人民币。人民币走强 → 换算回外币收益更高 → 外资流入更积极； 人民币走弱时反之。

观察窗口

  • USDCNY 汇率：升破某心理关口（如 7.3、7.5）时常伴随北向资金净流出
  • 北向资金日度净流入：沪深港通数据，可在东财、雪球直接查
  • 中美利差：10Y 中债 - 10Y 美债，利差收窄时外资撤离压力大

3. 央行的货币政策工具

工具作用影响板块
LPR（贷款市场报价利率）引导企业和个人贷款利率地产 / 金融 / 所有高负债行业
MLF（中期借贷便利）央行给银行的批发利率银行间流动性
RRR（存款准备金率）调节银行可贷资金信贷扩张速度
逆回购短期流动性投放A 股短期波动
PSL（抵押补充贷款）定向注入房地产等房地产板块

4. 三个最关键的宏观数据

CPI（消费者物价指数）

过低（<1%）暗示需求疲软、通缩风险；过高（>3%）压迫货币政策收紧。 CPI 破 2% 上行是国内货币政策可能转紧的信号。

PMI（采购经理人指数）

[0, 100] 区间。>50 扩张，<50 收缩。跌破 50常被称作"跌破枯荣线"。 官方 PMI 和财新 PMI 常被同时参考（前者偏大中型国企，后者偏中小民企）。

社融（社会融资规模）

实体经济从金融系统获得的全部资金。社融同比增速回落常预示经济放缓， 因为企业不再愿意借钱扩张。

三个数据看一眼就够 3 分钟：CPI 看通胀、PMI 看景气、社融看扩张意愿。

5. 政策周期速读

关键会议时间表

时间会议主要议题
每年 3 月两会 (全国人大+政协)政府工作报告 / GDP 目标 / 财政预算
每月中下旬LPR 报价贷款利率
每季末人民银行例行发布会货币政策回顾与展望
每年 12 月中央经济工作会议下一年度经济工作定调
每年 7/10 月政治局经济会议半年/秋季政策微调
每月美联储议息 (隔月开会)美国利率政策，全球外溢

政策"暗号"解读

  • "稳中求进" → 政策偏稳健，不搞大刺激
  • "逆周期调节" → 经济压力大，货币财政要发力
  • "房住不炒" → 地产相关紧缩信号持续
  • "稳预期、稳增长、稳就业" → 全面宽松前奏
